# Bài toán: Đồng hồ điện tử

**Time Limit:** 1.0s  |  **Memory Limit:** 1G

## Đề bài

Pink rất thích nhìn đồng hồ điện tử hiển thị theo dạng $HH:MM$, trong đó giờ từ $00$ đến $23$ và phút từ $00$ đến $59$.

Pink gọi một thời điểm là **"đồng điệu"** nếu tổng các chữ số của phần giờ bằng đúng tổng các chữ số của phần phút. 
*   Ví dụ: $14:32$ là thời điểm đồng điệu vì $1+4 = 3+2 = 5$; 
*   $07:07$ cũng là thời điểm đồng điệu vì $0+7 = 0+7 = 7$.

**Yêu cầu:** Hiện tại đồng hồ đang chỉ $H$ giờ $M$ phút. Hãy tính xem, tính cả thời điểm hiện tại, trong vòng $K$ phút tiếp theo có bao nhiêu thời điểm đồng hồ hiển thị thời gian đồng điệu. (Lưu ý: khoảng thời gian này có thể kéo dài sang ngày hôm sau hoặc nhiều ngày sau đó).

## Dữ liệu nhập vào từ bàn phím
Gồm ba dòng, mỗi dòng ghi một số tự nhiên, lần lượt là giá trị của $H, M$ và $K$ ($0 \le H \le 23; 0 \le M \le 59; 0 \le K \le 2880$).

## Kết quả ghi ra màn hình
Ghi ra một dòng chứa một số tự nhiên duy nhất là số thời điểm đồng điệu tìm được.

## Ví dụ

**Input 1**
```text
23
58
5
```

**Output 1**
```text
1
```

**Input 2**
```text
7
7
0
```

**Output 2**
```text
1
```

# Giải thích chi tiết bài toán "Đồng hồ điện tử"

## 1. Định nghĩa "Đồng điệu"
Một thời điểm $HH:MM$ được gọi là đồng điệu khi:
`chữ_số_hàng_chục_giờ + chữ_số_hàng_đơn_vị_giờ = chữ_số_hàng_chục_phút + chữ_số_hàng_đơn_vị_phút`

Ví dụ:
*   Giờ $23$: Tổng chữ số là $2 + 3 = 5$.
*   Phút $58$: Tổng chữ số là $5 + 8 = 13$.
*   $\rightarrow 5 \neq 13$, không đồng điệu.

## 2. Cách tiếp cận: Mô phỏng (Simulation)
Vì giá trị $K$ tối đa là $2880$ phút (tương đương đúng 2 ngày), chúng ta có thể dễ dàng mô phỏng từng phút một để kiểm tra.

### Các bước thực hiện:
1.  Bắt đầu từ thời điểm $(H, M)$.
2.  Lặp $K+1$ lần (vì tính cả thời điểm hiện tại):
    *   Tính tổng chữ số của $H$: `sum_h = (H // 10) + (H % 10)`.
    *   Tính tổng chữ số của $M$: `sum_m = (M // 10) + (M % 10)`.
    *   Nếu `sum_h == sum_m`, tăng biến đếm.
    *   Tăng thời gian lên 1 phút:
        *   $M = M + 1$.
        *   Nếu $M = 60$, thì $M = 0$ và $H = H + 1$.
        *   Nếu $H = 24$, thì $H = 0$.

## 3. Phân tích ví dụ 1
$(H=23, M=58, K=5)$:
*   Phút 0 (hiện tại): $23:58 \rightarrow 2+3=5, 5+8=13 \rightarrow$ Không.
*   Phút 1: $23:59 \rightarrow 2+3=5, 5+9=14 \rightarrow$ Không.
*   Phút 2: $00:00 \rightarrow 0+0=0, 0+0=0 \rightarrow$ **Đồng điệu!** (Đếm = 1)
*   Phút 3: $00:01 \rightarrow 0, 1 \rightarrow$ Không.
*   Phút 4: $00:02 \rightarrow 0, 2 \rightarrow$ Không.
*   Phút 5: $00:03 \rightarrow 0, 3 \rightarrow$ Không.
**Kết quả: 1.**

## Lời giải (Python)

```python
import sys

def sum_digits(n):
    return (n // 10) + (n % 10)

# Đọc dữ liệu
try:
    h = int(sys.stdin.readline())
    m = int(sys.stdin.readline())
    k = int(sys.stdin.readline())
except:
    pass

count = 0
# Lặp k + 1 lần để tính cả thời điểm hiện tại
for _ in range(k + 1):
    if sum_digits(h) == sum_digits(m):
        count += 1
    
    # Tăng 1 phút
    m += 1
    if m == 60:
        m = 0
        h += 1
        if h == 24:
            h = 0

print(count)
```

### Cách 2: Sử dụng vòng lặp `for` (Quy đổi ra tổng số phút)
Thay vì tự quản lý việc tăng phút và giờ theo cách thủ công, ta có thể quy đổi thời gian hiện tại ra tổng số phút trong ngày, sau đó dùng phép chia lấy dư để cập nhật thời gian.

```python
import sys

def sum_digits(n: int) -> int:
    return n // 10 + n % 10

def is_harmonious(h: int, m: int) -> bool:
    return sum_digits(h) == sum_digits(m)

try:
    h = int(sys.stdin.readline())
    m = int(sys.stdin.readline())
    k = int(sys.stdin.readline())
except:
    pass

current_minutes = h * 60 + m
harmonious_count = 0

for _ in range(k + 1):
    current_h = (current_minutes // 60) % 24
    current_m = current_minutes % 60
    
    if is_harmonious(current_h, current_m):
        harmonious_count += 1
        
    current_minutes = (current_minutes + 1) % 1440

print(harmonious_count)
```

### Cách 3: Sử dụng vòng lặp `while`
Ta có thể sử dụng vòng lặp `while True` và đếm số phút đã trôi qua. Cách này cần lưu ý kiểm tra điều kiện dừng (đã đủ $K$ phút chưa) ngay sau khi vừa đếm xong thời điểm hiện tại.

```python
import sys

def sum_digits(n: int) -> int:
    return n // 10 + n % 10

def is_harmonious(h: int, m: int) -> bool:
    return sum_digits(h) == sum_digits(m)

try:
    h = int(sys.stdin.readline())
    m = int(sys.stdin.readline())
    k = int(sys.stdin.readline())
except:
    pass

current_minutes = h * 60 + m
harmonious_count = 0
elapsed_minutes = 0

while True:
    current_h = (current_minutes // 60) % 24
    current_m = current_minutes % 60
    
    if is_harmonious(current_h, current_m):
        harmonious_count += 1
        
    if elapsed_minutes == k:
        break
        
    elapsed_minutes += 1
    current_minutes = (current_minutes + 1) % 1440

print(harmonious_count)
```
