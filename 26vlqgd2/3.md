# Bài toán: Quân hậu

**Time Limit:** 1.0s  |  **Memory Limit:** 1G

## Đề bài

Cho một bàn cờ vua kích thước $N \times N$. Các hàng được đánh số từ $1$ đến $N$ từ trên xuống dưới, các cột được đánh số từ $1$ đến $N$ từ trái sang phải.

Một quân hậu đang đứng ở ô $(x, y)$. Quân hậu được đi theo luật cờ vua, nghĩa là trong một nước đi, quân hậu có thể di chuyển đến bất kỳ ô nào cùng hàng, cùng cột hoặc cùng đường chéo với ô hiện tại, miễn là ô đó vẫn nằm trong bàn cờ.

**Yêu cầu:** Hãy đếm xem có bao nhiêu ô $(u, v)$ mà quân hậu có thể đi tới trong đúng 1 nước đi và thỏa mãn điều kiện tổng $u + v$ là số chẵn.

## Dữ liệu nhập vào từ bàn phím

Gồm ba dòng, mỗi dòng ghi một số tự nhiên, lần lượt là giá trị của $N, x$ và $y$ ($1 \le x, y \le N \le 10^8$).

## Kết quả ghi ra màn hình

Ghi ra một dòng chứa một số tự nhiên duy nhất là số cách chọn ô tiếp theo thỏa mãn yêu cầu.

## Ví dụ

**Input 1**
```text
8
4
4
```

**Output 1**
```text
19
```

**Input 2**
```text
5
2
1
```

**Output 2**
```text
5
```

# Phân tích chi tiết

## 1. Điều kiện di chuyển

Quân hậu tại $(x, y)$ có thể đi tới ô $(u, v)$ nếu ô đó khác ô hiện tại và thuộc một trong các trường hợp:

1. Cùng hàng: $u = x$.
2. Cùng cột: $v = y$.
3. Cùng đường chéo chính: $u - v = x - y$.
4. Cùng đường chéo phụ: $u + v = x + y$.

Ngoài ra, ô đến phải thỏa mãn:

```python
(u + v) % 2 == 0
```

Vì $N$ có thể lên tới $10^8$, ta không thể duyệt từng ô trên bàn cờ. Cần dùng công thức tính trực tiếp.

---

## 2. Đếm trên hàng và cột

### Cùng hàng $x$

Nếu quân hậu đi trên hàng $x$, ta có $u = x$.

Điều kiện tổng chẵn trở thành:

```text
x + v chẵn
```

Do đó $v$ phải cùng tính chẵn lẻ với $x$.

### Cùng cột $y$

Nếu quân hậu đi trên cột $y$, ta có $v = y$.

Điều kiện tổng chẵn trở thành:

```text
u + y chẵn
```

Do đó $u$ phải cùng tính chẵn lẻ với $y$.

Ta dùng hàm sau để đếm số số trong đoạn $[1, N]$ có cùng tính chẵn lẻ với một số $a$:

```python
def count_same_parity(a, n):
    if a % 2 == 1:
        return (n + 1) // 2
    return n // 2
```

Nếu `a` lẻ, hàm trả về số số lẻ từ `1` đến `n`. Nếu `a` chẵn, hàm trả về số số chẵn từ `1` đến `n`.

---

## 3. Đếm trên hai đường chéo

### Đường chéo chính

Đường chéo chính đi qua $(x, y)$ có dạng:

```text
u - v = x - y
```

Số ô trên đường chéo này là:

```python
main_diag = n - abs(x - y)
```

### Đường chéo phụ

Đường chéo phụ đi qua $(x, y)$ có dạng:

```text
u + v = x + y
```

Số ô trên đường chéo phụ là:

```python
if x + y <= n + 1:
    anti_diag = x + y - 1
else:
    anti_diag = 2 * n - x - y + 1
```

---

## 4. Điểm dễ sai: trường hợp $x + y$ lẻ

Đây là chỗ solution cũ dễ bị sai.

### Nếu $x + y$ chẵn

Ô hiện tại $(x, y)$ có tổng chẵn, nên khi đếm trên hàng, cột và hai đường chéo, ô hiện tại đều bị tính vào.

Nhưng quân hậu phải đi sang ô khác, không được đứng yên. Vì vậy mỗi hướng phải trừ đi ô hiện tại:

```python
ans += row_count - 1
ans += col_count - 1
ans += main_diag - 1
ans += anti_diag - 1
```

Hai đường chéo đều hợp lệ vì mọi ô trên hai đường chéo qua $(x, y)$ có cùng tính chẵn lẻ của tổng với $(x, y)$.

### Nếu $x + y$ lẻ

Ô hiện tại $(x, y)$ có tổng lẻ.

Trên hàng:

- Ta cần $v$ cùng chẵn lẻ với $x$.
- Nhưng vì $x + y$ lẻ nên $y$ khác chẵn lẻ với $x$.
- Do đó ô hiện tại không bị đếm vào hàng.

Trên cột:

- Ta cần $u$ cùng chẵn lẻ với $y$.
- Nhưng vì $x + y$ lẻ nên $x$ khác chẵn lẻ với $y$.
- Do đó ô hiện tại không bị đếm vào cột.

Hai đường chéo không đóng góp ô nào, vì mọi ô trên hai đường chéo qua $(x, y)$ đều có tổng lẻ.

Vì vậy trong trường hợp này:

```python
ans = row_count + col_count
```

Không được trừ `1`.

## 5. Ba cách cài đặt

### Cách 1: Duyệt toàn bộ bàn cờ - $O(N^2)$

Cách này dễ hiểu nhất: duyệt mọi ô `(u, v)`, kiểm tra ô đó có phải ô quân hậu đi được không, rồi kiểm tra `(u + v)` có chẵn không.

Cách này chỉ dùng để hiểu bài hoặc ăn test nhỏ. Với `N <= 10^8` thì không thể AC.

```python
n = int(input())
x = int(input())
y = int(input())

ans = 0

for u in range(1, n + 1):
    for v in range(1, n + 1):
        if u == x and v == y:
            continue

        same_row = (u == x)
        same_col = (v == y)
        same_diag = (abs(u - x) == abs(v - y))

        if same_row or same_col or same_diag:
            if (u + v) % 2 == 0:
                ans += 1

print(ans)
```

### Cách 2: Duyệt 4 đường đi của quân hậu - $O(N)$

Thay vì duyệt cả bàn cờ, ta chỉ duyệt:

- Hàng chứa quân hậu.
- Cột chứa quân hậu.
- Đường chéo chính.
- Đường chéo phụ.

Cách này nhanh hơn cách 1, nhưng vẫn chưa đủ tốt khi `N = 10^8`.

```python
n = int(input())
x = int(input())
y = int(input())

ans = 0

# Cùng hàng
for v in range(1, n + 1):
    if v != y and (x + v) % 2 == 0:
        ans += 1

# Cùng cột
for u in range(1, n + 1):
    if u != x and (u + y) % 2 == 0:
        ans += 1

# Đường chéo chính: u - v = x - y
for u in range(1, n + 1):
    v = u - (x - y)
    if 1 <= v <= n and u != x and (u + v) % 2 == 0:
        ans += 1

# Đường chéo phụ: u + v = x + y
for u in range(1, n + 1):
    v = x + y - u
    if 1 <= v <= n and u != x and (u + v) % 2 == 0:
        ans += 1

print(ans)
```

### Cách 3: Dùng công thức - $O(1)$

Đây là cách cần nộp để AC.

```python
import sys


def count_same_parity(a, n):
    if a % 2 == 1:
        return (n + 1) // 2
    return n // 2


data = list(map(int, sys.stdin.read().split()))
n, x, y = data[0], data[1], data[2]

row_count = count_same_parity(x, n)
col_count = count_same_parity(y, n)

if (x + y) % 2 == 0:
    ans = row_count - 1
    ans += col_count - 1

    main_diag = n - abs(x - y)
    ans += main_diag - 1

    if x + y <= n + 1:
        anti_diag = x + y - 1
    else:
        anti_diag = 2 * n - x - y + 1
    ans += anti_diag - 1
else:
    ans = row_count + col_count

print(ans)
```

## 6. Kiểm tra ví dụ

### Ví dụ 1

Với $N = 8, x = 4, y = 4$:

```text
x + y = 8
```

Đây là số chẵn.

- Hàng 4: các cột chẵn là `2, 4, 6, 8`, bỏ cột `4`, còn `3` ô.
- Cột 4: các hàng chẵn là `2, 4, 6, 8`, bỏ hàng `4`, còn `3` ô.
- Đường chéo chính có `8` ô, bỏ ô hiện tại, còn `7` ô.
- Đường chéo phụ có `7` ô, bỏ ô hiện tại, còn `6` ô.

Tổng:

```text
3 + 3 + 7 + 6 = 19
```

### Ví dụ 2

Với $N = 5, x = 2, y = 1$:

```text
x + y = 3
```

Đây là số lẻ.

- Hàng 2: cần cột chẵn, có `2, 4`, được `2` ô.
- Cột 1: cần hàng lẻ, có `1, 3, 5`, được `3` ô.
- Hai đường chéo không có ô nào có tổng chẵn.

Tổng:

```text
2 + 3 = 5
```
